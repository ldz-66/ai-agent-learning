# AI文案助手 需求文档

## 产品定位
帮助用户快速生成不同平台、不同风格的文案内容。

## 核心功能
1. 用户输入：主题/产品名、目标平台、文案风格、补充要求
2. AI生成：根据输入生成对应文案
3. 结果展示：流式显示生成过程，支持复制

## 支持的平台风格
- 小红书：活泼、emoji多、感叹号、"姐妹们/宝子们"
- 朋友圈：简短、温暖、有共鸣感
- 微博：简洁、有话题感、适合转发
- 产品描述：专业、突出卖点、有说服力
- 广告语：简短有力、朗朗上口

## 页面结构
- 左侧：输入区（主题、平台选择、风格要求、字数限制）
- 右侧：输出区（流式显示生成的文案、复制按钮）

## 接口设计
POST /generate
  请求体：{ topic, platform, requirements, max_length }
  返回：流式文本

GET /health
  返回：服务状态
