# 个人知识库问答系统

> 基于RAG（检索增强生成）技术的智能文档问答系统

## 项目截图
（如果能截图贴上来最好，GitHub支持直接在md里放图片）

## 功能
- 📄 上传文档：支持拖拽上传 .txt 文件
- 🔍 语义检索：基于Embedding向量，理解问题语义
- 💬 智能问答：结合检索结果生成回答，流式显示
- 📌 来源引用：每个回答自动标注信息来自哪份文档
- 🗑 文档管理：查看、删除文档，一键重置

## 技术栈
| 层级 | 技术 |
|------|------|
| AI生成 | DeepSeek API（通过openai库） |
| 文本向量化 | sentence-transformers（本地模型） |
| 向量数据库 | ChromaDB |
| 后端 | FastAPI + Python |
| 前端 | 原生HTML/CSS/JavaScript |
