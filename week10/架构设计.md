# 智能任务管理助手架构

## 组件
1. **数据库层**（db.py）：SQLite，存储任务数据
2. **工具层**（tools.py）：数据库操作封装成工具函数
3. **Agent层**（agent.py）：DeepSeek + Tool Use，理解意图并调用工具
4. **API层**（main.py）：FastAPI，提供HTTP接口
5. **前端层**（static/index.html）：聊天界面

## 数据库表设计
tasks 表：
- id：自增主键
- title：任务标题（必填）
- description：任务描述（可选）
- priority：优先级（high/medium/low，默认medium）
- done：是否完成（0/1，默认0）
- due_date：截止日期（可选，格式 YYYY-MM-DD）
- created_at：创建时间（自动）

## 工具清单
- create_task：创建任务
- get_tasks：查询任务列表（支持筛选）
- update_task_done：标记任务完成/未完成
- update_task_info：更新任务信息
- delete_task：删除任务
- get_task_stats：获取任务统计（今日完成数等）
